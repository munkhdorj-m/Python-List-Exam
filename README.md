# Python List Exam

---

## Exercise 1

**Problem:**

Write a function that returns the sum of all numbers in a list.

**Example:**

	Input: [8, 12, 5, 10]  
	Output: 35
	
	Input: [1, 1, 1, 1, 1]  
	Output: 5

---

## Exercise 2

**Problem:**

Write a function that returns the smallest number in a list.

**Example:**

	Input: [7, -4, 9, 2]  
	Output: -4
	
	Input: [100, 50, 25]  
	Output: 25

---

## Exercise 3

**Problem:**

Write a function that counts how many times a target number appears in a list.

**Example:**

	Input: [4, 2, 4, 7, 4], target=4  
	Output: 3
	
	Input: [1, 2, 3], target=5  
	Output: 0

---

## Exercise 4

**Problem:**

Move all zeroes to the end of the list while keeping the order of other elements.

**Example:**

	Input: [0, 4, 0, 2, 9]  
	Output: [4, 2, 9, 0, 0]
	
	Input: [1, 2, 3]  
	Output: [1, 2, 3]

---

## Exercise 5

**Problem:**

Return a list containing all indices (plural of index) where the target element appears.

**Example:**

	Input: [5, 1, 5, 3, 5], target=5  
	Output: [0, 2, 4]
	
	Input: [7, 8, 9], target=1  
	Output: []

---

## Exercise 6

Problem:

Create and return a new list that contains the original list followed by the same elements in reverse order.

Function:
mirror_list(lst)

Example:

	Input: [1, 2, 3]
	Output: [1, 2, 3, 3, 2, 1]
	
	Input: [5, 8]
	Output: [5, 8, 8, 5]
	
	Input: [7]
	Output: [7, 7]
